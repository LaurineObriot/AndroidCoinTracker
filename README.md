# AndroidCoinTracker

Projet personnel fait en 1 semaine.

Simple app to tracking price of Cryptocurrency with Coinmarketcap API and Kotlin.

Librairies :
- CardView
- OkHttp3 : to fetch API
- Picasso : To load Image
- Gson : to convert result
